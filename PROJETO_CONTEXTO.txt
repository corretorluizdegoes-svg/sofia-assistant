SEÇÃO 2 — ROTAS E PÁGINAS
================================================================================

Rotas definidas em src/App.tsx:

• src/pages/Landing.tsx
  - Rota: /
  - Propósito: Página inicial pública de marketing/apresentação da S.O.F.I.A.
  - Componentes principais: FlowBackground, LanguageSwitcher, SofiaAvatar3D, Button (shadcn)
  - Hooks/contexts: useTranslation (react-i18next)

• src/pages/SaibaMais.tsx
  - Rota: /saiba-mais
  - Propósito: Página explicativa sobre como o sistema funciona, módulos, níveis e CTA para signup.
  - Componentes principais: FlowBackground, Button, ícones de módulo/nível (sofia-icons)
  - Hooks/contexts: nenhum (usa dados estáticos modulos/niveis de sofia-data)

• src/pages/Auth.tsx
  - Rota: /auth (com query ?mode=signup|login)
  - Propósito: Tela de login/cadastro por email+senha e OAuth Google.
  - Componentes principais: FlowBackground, LanguageSwitcher, Button, Input, Label, SofiaAvatarIcon
  - Hooks/contexts: useAuth (AuthContext), useToast, useTranslation, useSearchParams
  - Integrações: supabase.auth.signUp / signInWithPassword, lovable.auth.signInWithOAuth("google")

• src/pages/Index.tsx
  - Rota: /app (protegida por ProtectedRoute)
  - Propósito: Tela principal — chat com Sofia + sidebars de currículo e progresso.
  - Componentes principais: ChatSofia, SidebarEsquerda, SidebarDireita, DisciplinaPanel,
    BottomNav, LanguageSwitcher, UserAvatar, Sheet
  - Hooks/contexts: useAuth, useProgresso, useProfile, useLanguagePreference,
    useCurriculoI18n, useTranslation, useNavigate

• src/pages/MapaMental.tsx
  - Rota: /mapa-mental (protegida)
  - Propósito: Mapa mental interativo (D3 force simulation) dos conhecimentos do usuário.
  - Componentes principais: StarField, LanguageSwitcher, Button, Input, Dialog, EdgeCard (interno)
  - Hooks/contexts: useMapaMental, useTranslation, useCurriculoI18n
  - Integrações: Edge Function sofia-explicar-conexao para gerar texto de pontes IA.

• src/pages/Perfil.tsx
  - Rota: /perfil (protegida)
  - Propósito: Visualização e edição do perfil do usuário (avatar, codinome, redes, visibilidade)
    + estatísticas de XP/nível/conquistas.
  - Componentes principais: UserAvatar, Button, Input, Textarea, Switch, Label
  - Hooks/contexts: useAuth, useProfile, useProgresso, useToast, useTranslation

• src/pages/NotFound.tsx
  - Rota: * (catch-all)
  - Propósito: Página 404 padrão. Loga no console a rota inválida.
  - Componentes principais: <a>, divs base
  - Hooks/contexts: useLocation, useEffect

SEÇÃO 4 — CONTEXTS E HOOKS
================================================================================

src/contexts/

• AuthContext.tsx → AuthProvider + useAuth()
  - Provê: { user: User|null, session: Session|null, loading: boolean, signOut: () => Promise<void> }
  - Estado interno: user, session, loading. Listener supabase.auth.onAuthStateChange + getSession.

src/hooks/

• use-mobile.tsx → useIsMobile(): boolean
  - Detecta viewport mobile via media query (breakpoint 768px).

• use-toast.ts → useToast() / toast()
  - Sistema de toasts (re-export do shadcn/ui).

• useConversas.ts → useConversas()
  - Provê: conversas[], conversaAtual, mensagens[], loading + funções
    criarConversa, trocarConversa, renomearConversa, apagarConversa,
    salvarMensagem, setMensagensLocal, recarregar.
  - Estado: lista de conversations + messages do usuário (Supabase).

• useLanguagePreference.ts → useLanguagePreference()
  - Sincroniza profiles.language ↔ i18next (carrega ao logar, persiste ao mudar).
  - Retorna: { current: string, changeLanguage: (code: string) => Promise<void> }.

• useMapaMental.ts → useMapaMental()
  - Provê: nodes, edges, loading, emergingField + ações updateNodePosition,
    commitNodeMove, addNode, addEdge, removeEdge, undo, redo, canUndo, canRedo.
  - Estado: grafo persistido em mind_map_nodes / mind_map_edges, fila de save
    debounced, histórico undo/redo, detecção de campo emergente.

• useProfile.ts → useProfile()
  - Provê: profile, loading, completo, nome, iniciais, corFundo,
    salvar(patch), uploadAvatar(file), recarregar.
  - Estado: row de profiles do usuário, lock contra duplicidade de codinome.

• useProgresso.ts → useProgresso()
  - Provê: xp, nivelAtual, proximoNivel, patenteAtual, progressoAteProximo,
    topicosExplorados, conquistas, totalMensagens, streakDias, dataAlistamento,
    ganharXP(tipo, metadata), registrarInteracao(mensagem), carregado, isMaxLevel.
  - Estado: row de user_progress (Supabase) + cache local.

SEÇÃO 5 — INTEGRAÇÕES BACKEND
================================================================================

Cliente Supabase
----------------
• Localização: src/integrations/supabase/client.ts (gerado automaticamente — não editar).
• Configuração: createClient<Database>(VITE_SUPABASE_URL, VITE_SUPABASE_PUBLISHABLE_KEY)
  com auth { storage: localStorage, persistSession: true, autoRefreshToken: true }.
• Tipos: src/integrations/supabase/types.ts (gerado automaticamente).

Tabelas (schema public)
-----------------------
• profiles
  - user_id (uuid, FK auth.users), display_name, codinome (único), avatar_url, bio,
    cidade, social_x/instagram/facebook/linkedin, publico (bool), language (text),
    created_at, updated_at.

• conversations
  - id, user_id, title, disciplina, modulo_id, created_at, updated_at.

• messages
  - id, conversation_id (FK conversations), user_id, role (user|assistant),
    content (text), created_at.

• mind_map_nodes
  - id, user_id, node_key (único por user), label, modulo_id, glow_color,
    descricao, is_custom (bool), x, y, created_at, updated_at.

• mind_map_edges
  - id, user_id, source_key, target_key, is_custom (bool), created_at.

• user_progress
  - id, user_id, xp (int), topicos_explorados (text[]), conquistas (text[]),
    total_mensagens (int), streak_dias (int), data_alistamento (date),
    ultimo_dia_ativo (date), created_at, updated_at.

• xp_events
  - id, user_id, tipo (text), xp_ganho (int), metadata (jsonb), created_at.

Funções RPC
-----------
• registrar_xp(_tipo text, _xp int, _metadata jsonb default '{}') → jsonb
• xp_para_level(_xp int) → int

Edge Functions (supabase/functions/)
------------------------------------
• sofia-chat — chat principal com a IA Sofia (LLM via Lovable AI Gateway, máx 500 chars,
  resumo da conversa anterior ao iniciar nova sessão).
• sofia-explicar-conexao — gera texto poético explicando a ponte entre dois conceitos
  no Mapa Mental.
• editor-chat — Modo Editor (admin) — responde sobre estrutura do projeto,
  exporta JSONs do currículo e referencia o system prompt da Sofia.

Auth flow (resumo)
------------------
1. AuthProvider monta um onAuthStateChange listener e chama getSession() inicial.
2. ProtectedRoute checa { user, loading } e redireciona para /auth se ausente.
3. Auth.tsx oferece duas vias:
   - email/senha: supabase.auth.signUp({ emailRedirectTo: /app, data:{display_name} })
     ou supabase.auth.signInWithPassword.
   - Google OAuth: lovable.auth.signInWithOAuth("google", { redirect_uri: /app }).
4. Após login, useEffect em Auth.tsx redireciona automaticamente para /app.
5. signOut em AuthContext chama supabase.auth.signOut().

SEÇÃO 7 — CONTEÚDO PEDAGÓGICO (S.O.F.I.A.)
================================================================================

Localização do conteúdo
-----------------------
HARDCODED em src/lib/sofia-data.ts (não vem do Supabase). O Supabase só guarda o
PROGRESSO do usuário (XP, conquistas, nós/arestas customizadas do mapa, conversas).

Schema dos dados de aprendizado
-------------------------------
type Disciplina = {
  id: string;                  // ex: "calculo_1"
  nome: string;                // PT canônico
  descricaoSimbolica: string;  // texto poético/explicativo
  conexoes: string[];          // nomes de outras disciplinas (PT)
}

type Modulo = {
  id: string;                  // ex: "matematica"
  nome: string;                // PT
  emoji: string;
  nivelEstrategico: string;    // "Fundação" | "Estrutura" | "Inteligência" | "Interpretabilidade" | "Fronteira"
  disciplinas: Disciplina[];
}

type Nivel = {
  nivel: number;          // 1..N
  nome: string;
  emoji: string;
  descricao: string;
  xpNecessario: number;
}

type Patente = {
  id: "associado" | "aprendiz" | "companheiro" | "mestre" | "comandante";
  nome: string; cor: string; glow: string;
  levels: [number, number];
}

5 módulos / "territórios"
-------------------------
1. matematica          📐 Fundação           (8 disciplinas)
2. computacao          💻 Estrutura          (6 disciplinas)
3. inteligencia_artificial 🤖 Inteligência    (6 disciplinas)
4. computacao_simbolica    🧠 Interpretabilidade (6 disciplinas)
5. fisica_quantica     ⚛️ Fronteira          (5 disciplinas)

Outros artefatos
----------------
• modulos[] — array completo dos módulos/disciplinas.
• niveis[] — patentes/níveis com XP necessário.
• convergencias — listas de cruzamentos entre disciplinas (campos emergentes).
• conquistasDisponiveis — lista de conquistas (id, nome, descricao).
• palavrasChavePorModulo — dicionário palavra-chave→módulo (detecta tópicos no chat).

Mapa mental: o "currículo base" é construído por src/lib/mapa-mental-base.ts
(construirMapaBase()) a partir de modulos + convergencias e seedado em
mind_map_nodes/mind_map_edges na primeira vez que o usuário abre o mapa.
A partir daí o usuário pode adicionar nós/arestas custom (is_custom=true).

### src/App.tsx

import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { BrowserRouter, Route, Routes } from "react-router-dom";
import { Toaster as Sonner } from "@/components/ui/sonner";
import { Toaster } from "@/components/ui/toaster";
import { TooltipProvider } from "@/components/ui/tooltip";
import { AuthProvider } from "@/contexts/AuthContext";
import { ProtectedRoute } from "@/components/ProtectedRoute";
import { EditorMode } from "@/components/editor-mode";
import Landing from "./pages/Landing";
import SaibaMais from "./pages/SaibaMais";
import Auth from "./pages/Auth";
import Index from "./pages/Index";
import MapaMental from "./pages/MapaMental";
import Perfil from "./pages/Perfil";
import NotFound from "./pages/NotFound";

const queryClient = new QueryClient();

const App = () => (
  <QueryClientProvider client={queryClient}>
    <TooltipProvider>
      <Toaster />
      <Sonner />
      <BrowserRouter>
        <AuthProvider>
          <Routes>
            <Route path="/" element={<Landing />} />
            <Route path="/saiba-mais" element={<SaibaMais />} />
            <Route path="/auth" element={<Auth />} />
            <Route
              path="/app"
              element={
                <ProtectedRoute>
                  <Index />
                </ProtectedRoute>
              }
            />
            <Route
              path="/mapa-mental"
              element={
                <ProtectedRoute>
                  <MapaMental />
                </ProtectedRoute>
              }
            />
            <Route
              path="/perfil"
              element={
                <ProtectedRoute>
                  <Perfil />
                </ProtectedRoute>
              }
            />
            <Route path="*" element={<NotFound />} />
          </Routes>
          <EditorMode />
        </AuthProvider>
      </BrowserRouter>
    </TooltipProvider>
  </QueryClientProvider>
);

export default App;

import { useEffect, useMemo, useRef, useState } from "react";

type EdgeCardState = {

const EDGE_FUN_URL = `${import.meta.env.VITE_SUPABASE_URL}/functions/v1/sofia-explicar-conexao`;

export default function MapaMental() {
  const { t, i18n } = useTranslation();
  const curr = useCurriculoI18n();
  const svgRef = useRef<SVGSVGElement>(null);
  const gRef = useRef<SVGGElement>(null);
  const simRef = useRef<d3.Simulation<MapNode, undefined> | null>(null);
  const {
    nodes,
    edges,
    loading,
    emergingField,
    updateNodePosition,
    commitNodeMove,

    removeEdge,

    canUndo,
    canRedo,
  } = useMapaMental();

  const [selected, setSelected] = useState<MapNode | null>(null);
  const [mode, setMode] = useState<Mode>("idle");
  const [pendingSource, setPendingSource] = useState<string | null>(null);
  const [edgeCard, setEdgeCard] = useState<EdgeCardState | null>(null);
  const [adding, setAdding] = useState(false);
  const [newLabel, setNewLabel] = useState("");

  // refs sempre atuais p/ handlers do d3
  const modeRef = useRef(mode);
  const pendingRef = useRef(pendingSource);
  useEffect(() => { modeRef.current = mode; }, [mode]);
  useEffect(() => { pendingRef.current = pendingSource; }, [pendingSource]);

  // Tradução de label baseado no nó
  function nodeLabel(n: MapNode): string {
    if (n.is_custom) return n.label;
    if (n.id.startsWith("mod:")) return curr.moduloNome(n.modulo_id, n.label);
    if (n.id.startsWith("conv:")) return curr.convergencia(n.label);
    return curr.disciplinaNome(n.label);
  }

  // ─── Inicializa simulação D3 ───────────────────────────────
  useEffect(() => {
    if (loading || !svgRef.current || !gRef.current) return;
    const svg = d3.select(svgRef.current);
    const g = d3.select(gRef.current);

    // Defs com gradientes por aresta
    let defs = g.select<SVGDefsElement>("defs");
    if (defs.empty()) defs = g.append("defs");
    defs.selectAll("*").remove();

    // Zoom + pan
    const zoom = d3
      .zoom<SVGSVGElement, unknown>()
      .scaleExtent([0.15, 4])
      .on("zoom", (event) => {
        g.attr("transform", event.transform.toString());
      });
    svg.call(zoom);
    const w = svgRef.current.clientWidth;
    const h = svgRef.current.clientHeight;
    svg.call(zoom.transform, d3.zoomIdentity.translate(w / 2, h / 2).scale(0.7));

    // ── Cópias internas ──
    const simNodes: MapNode[] = nodes.map((n) => ({ ...n }));
    const simLinks = edges
      .map((e) => ({
        source: typeof e.source === "string" ? e.source : e.source.id,
        target: typeof e.target === "string" ? e.target : e.target.id,
      }))
      .filter((l) => simNodes.find((n) => n.id === l.source) && simNodes.find((n) => n.id === l.target));

    // Cria um gradient por aresta
    simLinks.forEach((l, i) => {

      const gid = `lg-${i}`;

        .attr("gradientUnits", "userSpaceOnUse");
      grad.append("stop").attr("offset", "0%").attr("stop-color", s.glow_color).attr("stop-opacity", 0.85);
      grad.append("stop").attr("offset", "100%").attr("stop-color", t.glow_color).attr("stop-opacity", 0.85);
      (l as unknown as { gid: string }).gid = gid;
    });

    // ── Force simulation: muito mais espaçoso ──
    const sim = d3
      .forceSimulation<MapNode>(simNodes)
      .force(
        "link",
        d3
          .forceLink<MapNode, { source: string | MapNode; target: string | MapNode }>(simLinks)
          .id((d) => d.id)
          .distance(280)        // ↑ distância
          .strength(0.08),      // ↓ atração
      )
      .force("charge", d3.forceManyBody().strength(-900))   // ↑ repulsão
      .force("collide", d3.forceCollide<MapNode>().radius(70))
      .force("x", d3.forceX(0).strength(0.02))
      .force("y", d3.forceY(0).strength(0.02))
      .alpha(0.9)
      .alphaDecay(0.025);
    simRef.current = sim;

    // ── Render links ──
    const linkSel = g
      .select<SVGGElement>(".links")
      .selectAll<SVGLineElement, { source: MapNode; target: MapNode; gid: string }>("line")
      .data(
        simLinks as unknown as { source: MapNode; target: MapNode; gid: string }[],
        (d) => {
          const s = typeof d.source === "string" ? d.source : (d.source as MapNode).id;
          const t = typeof d.target === "string" ? d.target : (d.target as MapNode).id;
          return `${s}|${t}`;
        },
      )
      .join("line")
      .attr("stroke", (d) => `url(#${d.gid})`)
      .attr("stroke-width", 1.5)
      .attr("stroke-linecap", "round")
      .style("cursor", "pointer")
      .on("click", (event, d) => {
        event.stopPropagation();
        const s = typeof d.source === "string" ? d.source : (d.source as MapNode).id;
        const tt = typeof d.target === "string" ? d.target : (d.target as MapNode).id;
        // posição do clique em coordenadas de tela
        const px = event.clientX;
        const py = event.clientY;
        openEdgeCard(s, tt, px, py);
      })
      .on("mouseenter", function () {
        d3.select(this).attr("stroke-width", 2.5);
      })
      .on("mouseleave", function () {
        d3.select(this).attr("stroke-width", 1.5);
      });

    // ── Render nodes ──
    const nodeSel = g
      .select<SVGGElement>(".nodes")
      .selectAll<SVGGElement, MapNode>("g.node")
      .data(simNodes, (d) => d.id)
      .join((enter) => {
        const ge = enter.append("g").attr("class", "node").style("cursor", "pointer");
        ge.append("circle")
          .attr("class", "glow")
          .attr("r", 22)
          .attr("fill", (d) => d.glow_color)
          .attr("opacity", 0.18)
          .style("filter", "blur(10px)");
        ge.append("circle")
          .attr("class", "core")
          .attr("r", (d) => (d.modulo_id === "custom" ? 5 : d.id.startsWith("mod:") ? 9 : 6))
          .attr("fill", (d) => d.glow_color)
          .attr("stroke", "rgba(255,255,255,0.7)")
          .attr("stroke-width", 0.6);
        ge.append("text")
          .attr("class", "label")
          .attr("dy", 26)
          .attr("text-anchor", "middle")
          .attr("font-size", 11)
          .attr("font-family", "Plus Jakarta Sans, system-ui")
          .attr("fill", "rgba(255,255,255,0.78)")
          .attr("pointer-events", "none");
        return ge;
      });

    // labels (re-aplica em mudança de idioma)
    nodeSel.select<SVGTextElement>("text.label").text((d) => nodeLabel(d));

    // Click — selecionar / conectar
    nodeSel.on("click", (event, d) => {
      event.stopPropagation();
      if (modeRef.current === "connecting") {
        if (pendingRef.current && pendingRef.current !== d.id) {
          addEdge(pendingRef.current, d.id);
          setPendingSource(null);
          setMode("idle");
          return;
        }
        setPendingSource(d.id);
        return;
      }
      setSelected(d);
      setEdgeCard(null);
    });

    // destaque do nó pendente (modo conectar)
    nodeSel.select<SVGCircleElement>("circle.glow")
      .attr("opacity", (d) => (pendingSource === d.id ? 0.55 : 0.18));

    // Drag
    const drag = d3
      .drag<SVGGElement, MapNode>()
      .on("start", (event, d) => {
        if (!event.active) sim.alphaTarget(0.3).restart();
        d.fx = d.x;
        d.fy = d.y;
        // guarda posição inicial p/ histórico (undo)
        (d as unknown as { __from?: { x: number; y: number } }).__from = {
          x: d.x ?? 0,
          y: d.y ?? 0,
        };
      })
      .on("drag", (event, d) => {
        d.fx = event.x;
        d.fy = event.y;
      })
      .on("end", (event, d) => {
        if (!event.active) sim.alphaTarget(0);
        d.fx = null;
        d.fy = null;
        if (typeof d.x === "number" && typeof d.y === "number") {
          const from = (d as unknown as { __from?: { x: number; y: number } }).__from;
          if (from) {
            commitNodeMove(d.id, from, { x: d.x, y: d.y });
          } else {
            updateNodePosition(d.id, d.x, d.y);
          }
        }
      });
    nodeSel.call(drag);

    // Tick
    sim.on("tick", () => {
      linkSel
        .attr("x1", (d) => (d.source as MapNode).x ?? 0)
        .attr("y1", (d) => (d.source as MapNode).y ?? 0)
        .attr("x2", (d) => (d.target as MapNode).x ?? 0)
        .attr("y2", (d) => (d.target as MapNode).y ?? 0);
      nodeSel.attr("transform", (d) => `translate(${d.x ?? 0},${d.y ?? 0})`);
    });

    svg.on("click", () => {
      setSelected(null);
      setEdgeCard(null);
      if (modeRef.current === "connecting") {
        setMode("idle");
        setPendingSource(null);
      }
    });

    return () => {
      sim.stop();
    };
    // re-cria quando mudam nós/arestas; idioma muda só os labels via outro effect
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [loading, nodes.length, edges.length]);

  // Atualiza labels quando idioma muda (sem recriar simulação)
  useEffect(() => {
    if (!gRef.current) return;

      .select(".nodes")
      .selectAll<SVGTextElement, MapNode>("g.node text.label")

  }, [i18n.language, nodes]);

  // Destaca o nó pendente sem re-criar
  useEffect(() => {
    if (!gRef.current) return;
    d3.select(gRef.current)
      .select(".nodes")
      .selectAll<SVGCircleElement, MapNode>("g.node circle.glow")
      .attr("opacity", (d) => (pendingSource === d.id ? 0.55 : 0.18));
  }, [pendingSource]);

  // Tecla Escape
  useEffect(() => {
    function onKey(e: KeyboardEvent) {
      if (e.key === "Escape") {
        setSelected(null);
        setEdgeCard(null);
        setMode("idle");
        setPendingSource(null);
      }
      // Undo / Redo
      const meta = e.ctrlKey || e.metaKey;
      if (meta && e.key.toLowerCase() === "z" && !e.shiftKey) {
        e.preventDefault();
        void undo();
      } else if (meta && (e.key.toLowerCase() === "y" || (e.shiftKey && e.key.toLowerCase() === "z"))) {
        e.preventDefault();
        void redo();
      }
    }
    window.addEventListener("keydown", onKey);
    return () => window.removeEventListener("keydown", onKey);
  }, [undo, redo]);

  // ─── Texto da conexão ────────────────────────────────────
  function bridgeFromCurriculum(aLabel: string, bLabel: string): string | null {
    // intersecção das listas de "conexões" das duas disciplinas (PT)
    const A = encontrarDisciplina(aLabel)?.disciplina;
    const B = encontrarDisciplina(bLabel)?.disciplina;
    if (!A || !B) return null;
    const inter = A.conexoes.filter((c) => B.conexoes.includes(c));
    const ponteOriginal = inter[0];
    if (ponteOriginal) {
      const ponte = curr.disciplinaNome(ponteOriginal);
      const aT = curr.disciplinaNome(aLabel);
      const bT = curr.disciplinaNome(bLabel);
      // Texto base em PT, traduzido textual a partir do i18next quando útil
      const fromTo = t("mindMap.edgeFromTo", { a: aT, b: bT });
      // Mensagem simples e poética; quando necessário, IA gera versão melhor — mas evita custo aqui.
      return `${fromTo} — ${ponte}.`;
    }
    return null;
  }

  async function openEdgeCard(sId: string, tId: string, x: number, y: number) {
    const sNode = nodes.find((n) => n.id === sId);
    const tNode = nodes.find((n) => n.id === tId);
    if (!sNode || !tNode) return;
    const aLabel = nodeLabel(sNode);
    const bLabel = nodeLabel(tNode);

    setEdgeCard({ s: sId, t: tId, x, y, loading: true, text: "" });

    // 1) Tenta heurística da grade (sem custo)
    let text: string | null = null;
    if (!sNode.is_custom && !tNode.is_custom) {
      text = bridgeFromCurriculum(sNode.label, tNode.label);
    }

    // 2) Se nada encontrado OU pelo menos um é custom: chama IA
    if (!text) {
      try {
        const r = await fetch(EDGE_FUN_URL, {
          method: "POST",
          headers: {
            "Content-Type": "application/json",
            Authorization: `Bearer ${import.meta.env.VITE_SUPABASE_PUBLISHABLE_KEY}`,
          },
          body: JSON.stringify({
            source: aLabel,
            target: bLabel,
            language: i18n.language,
          }),
        });
        const data = await r.json();
        text = data?.text ?? "";
      } catch (err) {
        console.error("explicar-conexao error", err);
        text = "";
      }
    }

    setEdgeCard((curr) => curr ? { ...curr, loading: false, text: text ?? "" } : curr);
  }

  // Conexões do nó selecionado (labels já traduzidos)
  const selectedConexoes = useMemo(() => {
    if (!selected) return [] as string[];
    const out: string[] = [];
    for (const e of edges) {
      const s = typeof e.source === "string" ? e.source : e.source.id;
      const tt = typeof e.target === "string" ? e.target : e.target.id;
      const otherId = s === selected.id ? tt : tt === selected.id ? s : null;
      if (!otherId) continue;
      const n = nodes.find((x) => x.id === otherId);
      if (n) out.push(nodeLabel(n));
    }

function EdgeCard({
  state,
  colors,
  onClose,
  onDelete,
  getLabel,
}: {
  state: EdgeCardState;
  colors: { a: MapNode; b: MapNode };
  onClose: () => void;
  onDelete: () => void;
  getLabel: (id: string) => string;
}) {
  const { t } = useTranslation();
  const ref = useRef<HTMLDivElement>(null);

  // Posiciona próximo ao clique sem sair da viewport
  const W = 360;
  const H = 220;
  const padding = 16;
  const left = Math.min(Math.max(padding, state.x - W / 2), window.innerWidth - W - padding);
  const top = Math.min(Math.max(padding, state.y - H - 12), window.innerHeight - H - padding);

  useEffect(() => {
    function onDown(e: MouseEvent) {
      if (ref.current && !ref.current.contains(e.target as Node)) onClose();
    }
    window.addEventListener("mousedown", onDown);
    return () => window.removeEventListener("mousedown", onDown);
  }, [onClose]);

  const aColor = colors.a.glow_color;
  const bColor = colors.b.glow_color;

  return (
    <div
      ref={ref}
      className="absolute z-30 animate-fade-in"
      style={{ left, top, width: W }}
      onClick={(e) => e.stopPropagation()}
    >
      <div
        className="rounded-3xl p-[1px]"
        style={{ background: `linear-gradient(135deg, ${aColor}, ${bColor})` }}
      >
        <div className="rounded-3xl bg-black/80 backdrop-blur-xl p-4 text-white">
          <div className="flex items-center justify-between gap-3 mb-2">
            <div className="flex items-center gap-2 min-w-0">
              <span className="w-2 h-2 rounded-full" style={{ background: aColor, boxShadow: `0 0 8px ${aColor}` }} />
              <span className="text-xs text-white/80 truncate">{getLabel(state.s)}</span>
              <span className="text-white/30 text-xs px-1">→</span>
              <span className="w-2 h-2 rounded-full" style={{ background: bColor, boxShadow: `0 0 8px ${bColor}` }} />
              <span className="text-xs text-white/80 truncate">{getLabel(state.t)}</span>
            </div>
            <button onClick={onClose} className="text-white/40 hover:text-white shrink-0">
              <X className="w-4 h-4" />
            </button>
          </div>

          <p className="text-[13px] leading-relaxed text-white/85 min-h-[3.2em]">
            {state.loading ? (
              <span className="text-white/55 italic">{t("mindMap.edgeCardLoading")}</span>
            ) : state.text ? (
              state.text
            ) : (
              "—"
            )}
          </p>

          <div className="mt-3 flex justify-end">
            <button
              onClick={onDelete}
              className="text-[11px] text-red-300/80 hover:text-red-200 rounded-full px-2 py-1"
            >
              {t("common.delete")}
            </button>
          </div>
        </div>
      </div>
    </div>
  );
}

import { createContext, ReactNode, useContext, useEffect, useState } from "react";
import { Session, User } from "@supabase/supabase-js";
import { supabase } from "@/integrations/supabase/client";

type AuthContextValue = {
  user: User | null;
  session: Session | null;
  loading: boolean;
  signOut: () => Promise<void>;
};

const AuthContext = createContext<AuthContextValue | undefined>(undefined);

export function AuthProvider({ children }: { children: ReactNode }) {
  const [user, setUser] = useState<User | null>(null);
  const [session, setSession] = useState<Session | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Listener PRIMEIRO
    const { data: sub } = supabase.auth.onAuthStateChange((_evt, sess) => {
      setSession(sess);
      setUser(sess?.user ?? null);
    });

    // Depois recupera sessão existente
    supabase.auth.getSession().then(({ data: { session: sess } }) => {
      setSession(sess);
      setUser(sess?.user ?? null);
      setLoading(false);
    });

    return () => sub.subscription.unsubscribe();
  }, []);

  async function signOut() {
    await supabase.auth.signOut();
  }

  return (
    <AuthContext.Provider value={{ user, session, loading, signOut }}>
      {children}
    </AuthContext.Provider>
  );
}

export function useAuth() {
  const ctx = useContext(AuthContext);
  if (!ctx) throw new Error("useAuth must be used within AuthProvider");
  return ctx;
}

import { ReactNode } from "react";
import { Navigate } from "react-router-dom";
import { useAuth } from "@/contexts/AuthContext";

export function ProtectedRoute({ children }: { children: ReactNode }) {
  const { user, loading } = useAuth();

  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center bg-gradient-to-br from-[#e8f4fd] to-[#a8d8f0]">
        <div className="text-4xl animate-pulse">👩‍🏫</div>
      </div>
    );
  }

  if (!user) return <Navigate to="/auth" replace />;
  return <>{children}</>;
}

• MapaMental.tsx — A re-criação da simulação D3 depende de [loading, nodes.length,
  edges.length] (eslint-disable react-hooks/exhaustive-deps). Mudanças in-place
  em nodes/edges sem alterar o length não disparam re-render — se o conteúdo
  do nó mudar (ex.: glow_color ou descricao), pode não refletir até reload.

• Modo Editor (src/components/editor-mode/) — marcado como "temporário" no aviso
  do próprio modal; deve ser removido em versão futura junto com a Edge Function
  editor-chat e a configuração em supabase/config.toml.

• src/integrations/lovable/index.ts — wrapper para OAuth. Em uma sessão anterior
  o login com Google apresentou falhas; o provider foi reconfigurado mas o fluxo
  deve ser revalidado em produção.

• useMapaMental.ts — fila de save (debounced) com flush em beforeunload; em caso
  de aba fechada abruptamente sem rede, escritas pendentes podem se perder.

• Auth.tsx (signup) — emailRedirectTo aponta para /app, mas se o e-mail exigir
  confirmação e o usuário abrir o link de outro device sem sessão, o redirect
  cairá novamente em /auth (comportamento esperado, mas vale UX explicitar).

• Página /perfil — não há remoção de avatar (apenas substituição via upload).

• NotFound.tsx — usa <a href> em vez de <Link> do react-router; recarrega a
  página em vez de SPA navigation.
